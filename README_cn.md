 # 引擎
 MCPluginBox(这里简称“MCPB”) 使用H5GG驱动（一个iOS内存修改器）
 # 代码
在编写时使用html+js编写MCPB的核心部分，并使用objective-c编写的一些函数来辅助某些功能。
 # 关于源代码
我们只公开一些函数的源代码。如果您对功能有更好的建议，欢迎PR到我们的仓库。您的建议有可能被用于MCPB。
